## 需要注意

- 这台机器直接执行 `npm -v` 会被 PowerShell 的脚本策略拦住。
- 不代表 `npm` 本身坏了，而是 PowerShell 优先执行了 `npm.ps1`。
- 解决方法很简单：在 PowerShell 里用 `npm.cmd` 即可。

示例：

```powershell
npm.cmd -v
npm.cmd run preview
```
