# 账号注册清单

这份清单按你要走的两种路线拆开：
- `最低上线版`：先把站做出来并公开访问
- `完整变现版`：开始做域名、统计、广告、分销

建议统一使用一个常用邮箱注册，后面管理起来最省事。

## 最低上线版

这几个账号建议先注册，越早越省事。

| 账号 | 是否必须 | 作用 | 建议何时注册 | 你要提前准备什么 |
| --- | --- | --- | --- | --- |
| GitHub | 必须 | 托管网页代码，给 Cloudflare Pages 自动部署 | 现在就注册 | 常用邮箱、手机号、稳定密码 |
| Cloudflare | 必须 | 免费托管静态网站，生成公开访问网址 | GitHub 后立即注册 | 常用邮箱、手机号 |
| 常用邮箱 | 必须 | 接收 GitHub、Cloudflare、广告审核邮件 | 如果还没准备专用邮箱，先准备 | 邮箱名尽量长期使用 |

### 最低上线版注册顺序

1. 先注册 `GitHub`
2. 再注册 `Cloudflare`
3. 用 `GitHub` 仓库接 `Cloudflare Pages`

### 最低上线版完成标准

- [ ] GitHub 账号可正常登录
- [ ] Cloudflare 账号可正常登录
- [ ] 已能创建一个公开仓库
- [ ] 已能用 Cloudflare Pages 连接 GitHub

## 完整变现版

如果你后面要挂广告和做长期运营，下面这些账号建议逐步补齐。

| 账号 | 是否必须 | 作用 | 建议何时注册 | 你要提前准备什么 |
| --- | --- | --- | --- | --- |
| 域名平台账号 | 强烈建议 | 购买独立域名，利于品牌和收录 | 网站初版能访问后 | 常用邮箱、支付方式 |
| 百度统计 | 建议 | 看访问量、来源关键词、页面表现 | 网站公开后尽快 | 网站地址 |
| 百度广告联盟 | 变现必须 | 审核通过后可挂广告代码赚钱 | 网站内容和隐私页完善后 | 身份证、银行卡、手机号、网站地址 |
| 分销平台账号 | 可选 | 推广服务器、AI 工具、会员服务赚佣金 | 有稳定流量后 | 实名资料、收款信息 |
| 友情链接资源 | 可选 | 交换链接提升收录和权重 | 网站公开后 | 站点介绍、站点地址 |

## 建议准备的实名材料

下面这些最好提前整理好，后面广告和收款审核会用到：

- 身份证
- 本人银行卡
- 可接收短信的手机号
- 常用邮箱
- 网站公开地址
- 站点名称
- 站点简介
- 隐私政策页面链接
- 免责声明页面链接

## 账号注册建议

- `GitHub` 和 `Cloudflare` 可以用同一个邮箱，后面最好用密码管理器统一保存。
- 域名尽量和站点主题相关，比如带 `ai`、`ollama`、`learn`、`hub` 这类关键词。
- 广告联盟审核前，页面里一定要有：`关于本站`、`隐私政策`、`免责声明`。
- 不要在站里放破解、盗版、外挂、违规下载资源，不然广告审核和收录都容易出问题。
